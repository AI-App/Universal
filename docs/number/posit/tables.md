Posit value tables
==================

The program "education/tables/edu_tables_posits" will generate the posit encodings and associated real values for reference. 
These tables are a great aid in understanding posit arithmetic and rounding.

```text
>:~/dev/universal/build$ education/tables/edu_tables_posits::
    Generate posit configurations
    -128         100101         111011 Sign : -1 Regime :   1 Exponent :     8 Fraction :        1 Value :             -128 11
```
